# 🌐 WebHub

WebHub adalah website edukasi yang menyediakan materi **Web Development** dan **Cyber Security** secara gratis, terstruktur, dan mudah dipahami. Materi disusun mulai dari tingkat dasar hingga lanjutan untuk membantu proses belajar secara mandiri.

---

## ✨ Fitur

- 📚 Artikel Web Development
- 🛡️ Artikel Cyber Security
- 🔍 Pencarian Artikel
- 📱 Responsive Design
- ⚡ Tampilan Modern
- 📖 Detail Artikel
- 📂 Struktur Materi yang Terorganisir

## 🚀 Menjalankan Project

### Clone Repository

```bash
git clone https://github.com/vnocyber/webhub.git
```

### Masuk ke Folder

```bash
cd webhub
```

### Jalankan

Buka file **index.html** menggunakan browser.

Atau gunakan **Live Server** di Visual Studio Code.
